# 页面转化优化

这是 `page-cro` 的中文本地化说明版。它不是逐句硬译，而是把原始 `SKILL.md` 整理成更适合中文用户直接执行的工作手册；需要逐段对照英文原版时，请同时查看同目录下的 `SKILL.md`。

## 适用场景

当用户想提升首页、落地页、功能页或其他营销页面的转化时使用。适合做页面诊断、问题优先级排序、改版建议和测试想法。

## 使用方式

- 先读取 `.agents/product-marketing-context.md`（或旧路径 `.claude/product-marketing-context.md`），避免重复提问并沿用已有定位信息。
- 围绕“初始评估”推进任务，并在需要时回看原版 `Initial Assessment` 小节获取更细颗粒度指引。
- 按原版 `CRO Analysis Framework` 小节推进，确保不遗漏该技能特有的方法、检查点或框架。
- 围绕“输出格式”推进任务，并在需要时回看原版 `Output Format` 小节获取更细颗粒度指引。
- 按原版 `Page-Specific Frameworks` 小节推进，确保不遗漏该技能特有的方法、检查点或框架。
