# 网站架构

这是 `site-architecture` 的中文本地化说明版。它不是逐句硬译，而是把原始 `SKILL.md` 整理成更适合中文用户直接执行的工作手册；需要逐段对照英文原版时，请同时查看同目录下的 `SKILL.md`。

## 适用场景

当用户想规划、重组或可视化网站的信息架构时使用。适合页面层级、导航、URL 结构、内链和 sitemap 设计。

## 使用方式

- 先读取 `.agents/product-marketing-context.md`（或旧路径 `.claude/product-marketing-context.md`），避免重复提问并沿用已有定位信息。
- 围绕“规划前”推进任务，并在需要时回看原版 `Before Planning` 小节获取更细颗粒度指引。
- 围绕“网站类型与切入点”推进任务，并在需要时回看原版 `Site Types and Starting Points` 小节获取更细颗粒度指引。
- 围绕“页面层级设计”推进任务，并在需要时回看原版 `Page Hierarchy Design` 小节获取更细颗粒度指引。
- 围绕“导航设计”推进任务，并在需要时回看原版 `Navigation Design` 小节获取更细颗粒度指引。
