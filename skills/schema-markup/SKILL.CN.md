# 结构化数据标记

这是 `schema-markup` 的中文本地化说明版。它不是逐句硬译，而是把原始 `SKILL.md` 整理成更适合中文用户直接执行的工作手册；需要逐段对照英文原版时，请同时查看同目录下的 `SKILL.md`。

## 适用场景

当用户想添加、修复或优化网站 schema markup 时使用。适合实现 JSON-LD、选择 schema 类型、校验和排查 rich results 问题。

## 使用方式

- 先读取 `.agents/product-marketing-context.md`（或旧路径 `.claude/product-marketing-context.md`），避免重复提问并沿用已有定位信息。
- 围绕“初始评估”推进任务，并在需要时回看原版 `Initial Assessment` 小节获取更细颗粒度指引。
- 围绕“核心原则”推进任务，并在需要时回看原版 `Core Principles` 小节获取更细颗粒度指引。
- 围绕“常见 Schema 类型”推进任务，并在需要时回看原版 `Common Schema Types` 小节获取更细颗粒度指引。
- 围绕“速查表”推进任务，并在需要时回看原版 `Quick Reference` 小节获取更细颗粒度指引。
